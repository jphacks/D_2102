# Frontend Project

## Setup



### Operating Environment
OS: MacOS BigSur 11.0.1
docker-compose: version 1.29.1, build c34c88b2



### Command
```
docker-compose up -d
```


## Usage



### Command
```
docker-compose exec server_app /bin/bash
./mvnw spring-boot:run
```
Access http://localhost:8889/ with API Tester.
