# Frontend Project

## Setup



### Operating Environment
OS: MacOS BigSur 11.0.1
docker-compose: version 1.29.1, build c34c88b2



### Command
```
docker-compose up -d
```

## Usage

### Command
```
docker-compose exec front_app sh
cd front_app
yarn start
```
Access http://localhost:3000/ with any browser.
